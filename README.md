
## Descrição dos Problemas

### Problema 1: Soma
- **Arquivo:** `problema1/soma.py`
- **Descrição:** Script para calcular a soma de uma lista de números.

### Problema 2: Fibonacci
- **Arquivo:** `problema2/fibonacci.py`
- **Descrição:** Script para gerar a sequência de Fibonacci até um número específico.

### Problema 3: Faturamento
- **Arquivo:** `problema3/faturamento.py`
- **Descrição:** Analisa o faturamento diário de uma distribuidora. Calcula o menor valor, o maior valor e o número de dias com faturamento superior à média mensal.
- **Arquivos de Dados:**
  - `dados.xml` (opcional): Dados de faturamento no formato XML.

### Problema 4: Valor de faturamento mensal de uma distribuidora
- **Arquivo:** `problema4/percentual_faturamento.py`
- **Descrição:** Calcula o percentual de faturamento por estado.

### Problema 5: Inverter String
- **Arquivo:** `problema5/inverter_string.py`
- **Descrição:** Script para inverter uma string fornecida pelo usuário.

## Requisitos

Certifique-se de ter Python instalado em sua máquina para executar os scripts.
